# Coffee Shop - Учебный проект
## Описание
Учебный проект по верстке сайта кофейни "Рево-кофе". Проект представляет собой семантическую HTML5 разметку по предоставленному макету.

## Технологии
- HTML5
- Семантическая верстка

## Особенности
- Использование семантических тегов (header, main, section, article, aside, footer)
- Чистая разметка без CSS-классов
- Адаптивные изображения с alt-текстами

## Структура
- Шапка с логотипом и баннером
- Основной контент с разделами:
  - Преимущества кофе
  - Каталог продукции
  - Подарочные наборы
  - Комбо-предложения
- Подвал сайта

## Цель проекта
Освоить принципы семантической HTML-разметки, научиться создавать корректную структуру
веб-страницы на основе готового макета. Получить навыки подключения внешних ресурсов, работы с
типографикой и валидации HTML-кода. Закрепить на практике работу с Git и публикацию проекта на
GitHub Pages.
